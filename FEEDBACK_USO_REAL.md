# Feedback de uso real

Anotações de fricção coletadas enquanto o Luiz usa o app de verdade durante sessões de poker. Não é changelog, não é roadmap — é matéria-prima crua pra decidir prioridades de polish, M5 ou M6.

Formato sugerido (livre):

```
## YYYY-MM-DD — contexto curto da sessão (stake, duração, lugar)

- Fricção: o que aconteceu, qual fluxo
- Esperado: o que você queria que acontecesse
- Quão chato: leve / médio / bloqueante
- Ideia: solução que veio à cabeça (opcional)
```

Não precisa ser polido. Frases soltas, abreviações, tudo serve. Polish vem na hora de virar issue/PR.
